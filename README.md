Поисковый движок
Данная программа производит поиск данных по запросу ввиде текста с расчётом релевантности. Поиск производится по текстовым файлам, содержимое которых индексируется для последующего их анализа релевантности запросу. При поиске расчитывается релевантность содержимого текстовых файлов относительно запроса, предоставляя возможность увидеть насколько содержимое файла актуально для запроса относительно других файлов

Программа была написана при помощи CLion с использованием CMake для сборки и Json файлов для конфигурации, передачи текстовых файлов, запросов и закрепления ответа на запросы

В файле requests.json записываются интересующие запросы

В файле answers.json предоставляется ответ. rank - релевантность

В файле config.json записываются файлы, по которым производится поиск. Если какой-либо файл не найден - программа сообщит об этом

В процессе создания программы были написаны три ключевых класса и заголовочный файл для функций подсчёта количества слов, сортировку их на уникальные. В процессе работы программы текстовые файлы никак не меняются, лишь считывается их содержимое. Класс для передачи информации из Json файлов, файла конфигурации, запросов, ответов. Класс для индексации содержимого текстовых файлов - через который производится подсчёт всех слов, количество вхождений уникальных слов, здесь используются функции для посчёта, при этом содержимое текстовых файлов сохраняется внутри программы, переводя верхний регистр к нижнему для точного подсчёта уникальных слов. На данный момент программа работает лишь с латиницей

Подсчёт производится в нескольких потоках, обрабатывая несколько текстовых файлов сразу. Количество слов в файлах проверяется, программа работает только с файлами, количество слов в которых не превышает 1000, а в запросах - не более 10 слов. Также, одно слово должно содержать не более 100 символов. Слово для данной программы это набор букв латиницы, также программа проверяет наличие апострофа и учитывает его в слове, любые другие символы программа не учитывает

Для сборки программы:

Необходимо убедиться что у вас установлен набор инструментов для сборки программы, такой как Microsoft Software Development Kit, а также CMake - для возможности ввода команд для сборки

Используя системную консоль, необходимо перейти в директорию с исходными файлами программы

Находясь в указанной директории, достаточно ввести следующую команду:

cmake -S . -B build

-S . - указывает, что исходные файлы для сборки находятся в текущей директории

-B build - производит сборку программы в папке с наименованием "build", наименование может быть и другим, удобным вам. При отсутствии папки - она будет создана автоматически
